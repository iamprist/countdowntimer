##  Usage

### Timer Modes

1. **Countdown Timer**
   - Enter hours, minutes, and seconds
   - Click Start to begin countdown
   - Alarm sounds when timer reaches zero

2. **Count Up Timer**
   - Set a target time
   - Timer counts up from zero to target
   - Alarm sounds when target is reached

3. **Stopwatch** (Default)
   - Click Start to begin
   - No time limit, counts up indefinitely
   - Perfect for tracking work sessions

4. **Clock**
   - Displays current time automatically
   - Toggle between 12-hour and 24-hour format

### Controls

- **Start** - Begin timer
- **Pause** - Pause timer (or press Spacebar)
- **Reset** - Reset timer to zero
- ** Settings** - Access timer modes and themes
- **‚õ∂ Fullscreen** - Enter fullscreen mode
